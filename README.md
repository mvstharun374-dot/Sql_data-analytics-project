# 📊 SQL Data Analytics Project

## 🚀 About the Project
This project shows how I used SQL to understand and analyze data.  
I worked with a dataset and wrote different queries to find useful information and patterns.

The main goal of this project is to show how SQL can be used for **real data analysis**, not just basic queries.

---

## 🎯 What I Did
- Explored the data to understand its structure  
- Analyzed different columns like categories and numbers  
- Found top and bottom records  
- Studied how data changes over time  
- Created useful insights from the data  

---

## 🔍 Exploratory Data Analysis (EDA)
In this step, I tried to understand the data.

I performed:
- Checking tables and columns  
- Looking at different categories (like product, customer, etc.)  
- Analyzing numbers like total, average  
- Finding top and bottom values  

👉 This step helped me understand the data clearly.

---

## 📈 Advanced Analysis
After EDA, I did deeper analysis:

- Found trends over time  
- Calculated running totals  
- Compared performance  
- Grouped data into segments  

👉 This step helped me get meaningful insights.
